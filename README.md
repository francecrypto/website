# FranceCrypto — Développement local

Minimal: installer les dépendances puis lancer le serveur de développement.

Prereqs: Node.js (v18+ recommandé) et pnpm installés.

Installation

```bash
pnpm install
```

Lancer en local

```bash
pnpm run dev
```

Accéder ensuite à http://localhost:3000
